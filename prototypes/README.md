# prototypes/

Free-form sandbox for quick, throwaway experiments and spikes -- one-off
notebooks, scratch scripts, half-baked ideas. No structure or schema is
enforced here; nothing in this directory is a tracked result.

When a prototype proves out, promote it: move reusable code into the
`crafter_rl` package and turn the experiment itself into a proper
`experiments/<date>-name/` (copy `experiments/_template/`). The promotion
workflow is documented separately (docs to come).
